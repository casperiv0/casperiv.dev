## Requirements

- Node.js installed with **version v18 or higher**
- Basic Node.js app

## Installation

Vitest is super simple to install. It can be installed via any node package manager such as [npm](https://npmjs.org), [yarn](https://yarnpkg.org) or [pnpm](https://pnpm.io). We will install [vitest](https://vitest.dev) as a `devDependency`.

```bash
npm install --save-dev vitest
# OR
yarn add -D vitest
# OR
pnpm add -D vitest
```

## Configuring the project

We will now add a simple command to our `package.json`. This will allow us to run `npm run test` in the command line.

```json {5}
{
  "name": "my-nodejs-app",
  "version": "0.0.0",
  "scripts": {
    "test": "vitest"
  }
}
```

## Adding a test file

Once we've updated our configuration, we can start adding test files, for example `maths.test.ts`. We import `test` and `expect` from `vitest`:

```ts title="maths.test.ts"
import { expect, test } from "vitest";

test("Math.sqrt()", () => {
  expect(Math.sqrt(4)).toBe(2);
  expect(Math.sqrt(144)).toBe(12);
  expect(Math.sqrt(2)).toBe(Math.SQRT2);
});
```

## Testing the application

To run these tests, simply run the `test` command we created earlier:

```bash --command=true
npm run test
```

This will show all passed and failed tests.

<Info style={{ margin: "1rem 0" }}>
  **Tip:** You can also view those tests in the browser! Simply create a new command, shown below:

```json {4}
{
  "scripts": {
    "test": "vitest",
    "test:ui": "vitest --ui"
  }
}
```

</Info>
